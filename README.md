# 微信公众号内容分析技能

一个帮你把公众号文章拆开揉碎了看的内容分析方法库。不是简单的"好/不好"评价，而是给出可量化的评分、可复用的结构拆解、可执行的改进建议。

适合做公众号运营、品牌内容、新媒体投放分析的人用。

## 这个技能能干什么

把一篇公众号文章丢给它，它会从几个角度拆：

- **标题**：属于哪种类型（资讯/悬念/数字/情感等），吸引力打分，有没有标题党嫌疑
- **结构**：开头怎么切入的，正文怎么铺的，结尾有没有引导互动，整体字数和阅读时长
- **可读性**：句子平均多长，专业术语密度，口语还是书面，图文节奏
- **主题与情感**：核心主题是什么，情绪是积极/中性/焦虑诱导，价值观传递

如果你能提供互动数据（阅读量、点赞、在看、评论、分享），它还会算互动率、和行业基准对比、找爆款规律。

如果你给一批文章（比如某个号最近 20 篇），它能看到内容矩阵、发布节奏、爆款共性这些更整体的东西。

```bash

git clone https://github.com/like041123/wechat-content-analysis-skill.git \
  ~/.trae/skills/wechat-content-analysis

# 或项目级安装
git clone https://github.com/like041123/wechat-content-analysis-skill.git \
  .trae/skills/wechat-content-analysis
```

装好之后，你直接丢文章链接或正文，助手会自动按这个技能的方法分析。

## 怎么用

**分析单篇文章**：粘贴 `mp.weixin.qq.com` 链接或正文，告诉它"分析一下"。它会抓取元数据，按四个维度逐项打分，最后给一张评分卡和 3 条可执行的改进建议。

**诊断某个账号**：提供该号近期 10-20 篇文章样本，它完成内容矩阵分析、识别爆款规律、给出定位清晰度和优化建议。

**对比多个账号**：给两个以上账号的样本和数据，它按统一维度出对比表和差异化建议。

## 几个注意点

- 微信互动数据有刷量可能，分析时会给数据可信度提示，不会盲目相信数字
- 不同行业的基准差异很大（财经号阅读量天然比教育号高），不会拿同一把尺子量所有人
- 建议不涉及诱导分享、刷量这类违反微信运营规范的操作
