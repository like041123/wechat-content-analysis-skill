# 评分标准与行业基准

这里放的是各维度的评分细则和行业基准数据。打分的时候对照这里，避免凭感觉给分。

## 一、标题评分细则（1-10 分）

| 分数区间 | 等级 | 判定标准 |
|---------|------|---------|
| 9-10 | 行业标杆 | 核心信息前置 + 情绪钩子 + 长度适宜 + 无标题党嫌疑；朋友圈折叠线前完整展示核心信息 |
| 7-8 | 优秀 | 信息明确，有吸引力，长度适中（16-22 字），关键词突出 |
| 5-6 | 合格 | 传达了主题但吸引力一般，或长度偏长/偏短 |
| 3-4 | 待优化 | 信息模糊、缺乏吸引力，或存在轻度标题党特征 |
| 1-2 | 严重不足 | 纯标题党（信息完全隐藏/严重夸张）、与正文不符、长度严重不当 |

### 标题类型识别指南

| 类型 | 特征 | 示例模式 | 适用场景 |
|------|------|---------|---------|
| 资讯型 | 直接陈述事实/新闻 | "XX公司完成B轮融资" | 行业新闻、官方公告 |
| 悬念型 | 隐藏关键信息引发好奇 | "这个做法，让营收翻了3倍" | 干货分享、案例拆解 |
| 数字型 | 含具体数字增强可信度 | "5个方法，提升转化率200%" | 盘点、教程、方法论 |
| 情感型 | 调动情绪共鸣 | "那个35岁被裁的中年人，后来怎样了" | 故事、人物、社会议题 |
| 对比型 | 制造反差冲突 | "月薪3千和月薪3万的文案区别" | 技能对比、认知升级 |
| 盘点型 | 整理合集降低阅读门槛 | "2024年最全的私域运营指南" | 年终总结、资源合集 |
| 痛点型 | 直击用户困境 | "为什么你的公众号阅读量越来越低？" | 问题诊断、解决方案 |

### 标题党特征检测清单
- [ ] 是否使用了极端词汇（震惊、吓傻、崩溃、绝了）
- [ ] 核心信息是否被故意隐藏（"这个""那件事""真相是"）
- [ ] 是否与正文内容严重不符
- [ ] 是否过度煽动焦虑或恐惧情绪
- [ ] 是否使用虚假数字或夸大承诺

检测到 2 项以上命中 → 标题党风险高，评分不超过 4 分。

## 二、结构评分细则（1-10 分）

| 分数区间 | 等级 | 判定标准 |
|---------|------|---------|
| 9-10 | 行业标杆 | 黄金开头（3秒内抓住注意力）+ 清晰小标题层级 + 图文节奏佳 + 强力CTA结尾 |
| 7-8 | 优秀 | 结构完整，逻辑清晰，有明确的开头-正文-结尾，小标题使用合理 |
| 5-6 | 合格 | 基本结构完整但某环节较弱（如开头平淡或无结尾引导） |
| 3-4 | 待优化 | 结构混乱，段落过长（>300字/段），缺乏小标题 |
| 1-2 | 严重不足 | 无结构可言，大段文字堆砌，无分段或无逻辑 |

### 开头策略评分
- **10分**：3秒内制造悬念/冲突/共鸣，读者有强烈继续阅读欲望
- **7-8分**：开头有吸引力但不够强烈
- **5-6分**：开头平淡但信息明确
- **3-4分**：开头冗长，迟迟未进入主题
- **1-2分**：开头与主题无关或严重跑题

### CTA（结尾引导）评分
- **10分**：自然过渡 + 明确引导 + 低门槛动作（如"评论区说说你的看法"）
- **7-8分**：有引导但稍显生硬
- **5-6分**：简单结尾，无明确引导
- **3-4分**：戛然而止，无结尾设计
- **1-2分**：结尾与内容脱节

## 三、可读性评分细则（1-10 分）

| 指标 | 建议范围 | 评分参考 |
|------|---------|---------|
| 平均句长 | 15-25 字 | <15字偏碎裂，>30字偏冗长 |
| 段落长度 | 80-150 字 | >300字阅读疲劳 |
| 小标题间隔 | 每 300-500 字 | 无小标题超过 500 字扣分 |
| 图文比 | 每 300-500 字配 1 图 | 纯文字超 800 字无图扣分 |
| 专业术语密度 | <15% | 视受众调整，面向大众应 <10% |

### 可读性综合评分
- **9-10分**：句式多样、段落适中、图文节奏佳、术语适度、留白舒适
- **7-8分**：整体流畅但某项略有不足
- **5-6分**：基本可读但存在长句/长段/术语过多等问题
- **3-4分**：阅读体验差，多处长句/长段/无图
- **1-2分**：几乎无法顺畅阅读

## 四、互动数据行业基准

### 互动率基准（互动率 = (点赞+在看+评论+分享) / 阅读量）

| 账号层级 | 互动率范围 | 说明 |
|---------|-----------|------|
| 头部账号 | 3-8% | 粉丝 50万+，强粉丝粘性 |
| 中腰部账号 | 1-3% | 粉丝 5-50万，正常水平 |
| 长尾账号 | < 1% | 粉丝 <5万，或内容缺乏互动设计 |
| 爆款阈值 | > 账号均值 2 倍 | 单篇互动率显著高于日常 |

### 分项互动率参考

| 指标 | 健康范围 | 异常信号 |
|------|---------|---------|
| 点赞率 | 0.5-2% | <0.3% 内容缺乏认同感 |
| 在看率 | 0.3-1.5% | 在看率 ≈ 完读率代理指标 |
| 评论率 | 0.1-0.8% | >1% 说明话题引发强烈讨论 |
| 分享率 | 0.2-1% | >2% 具备传播价值（爆款潜力） |

### 阅读量行业参考（单篇）

| 行业 | 头部账号 | 中腰部账号 | 备注 |
|------|---------|-----------|------|
| 财经 | 10万+ | 1-5万 | 阅读量普遍偏高 |
| 科技 | 5-10万 | 5000-2万 | 受众活跃度高 |
| 教育 | 3-5万 | 3000-1万 | 家长群体为主 |
| 时尚美妆 | 5-10万 | 5000-2万 | 视觉内容占比高 |
| 本地生活 | 1-3万 | 1000-5000 | 地域属性强 |
| 企业官号 | 5000-2万 | 500-3000 | 受限于粉丝基数 |

> **注意**：以上基准为参考值，实际表现受账号粉丝量、内容质量、发布时间、平台推荐等多因素影响。2026 年微信推荐机制改版后，公域推荐流量占比提升，新号也可能获得高阅读量。

## 五、内容策略诊断基准

### 发布频率健康度

| 频率 | 评级 | 说明 |
|------|------|------|
| 日更 5-7 篇/周 | 高强度 | 适合资讯类，需注意质量稳定性 |
| 周更 3-5 篇/周 | 健康 | 适合大多数账号 |
| 周更 1-2 篇/周 | 维持 | 适合深度内容账号 |
| 月更 <4 篇/月 | 不足 | 粉丝活跃度下降风险高 |

### 黄金发布时段

| 时段 | 推荐内容类型 | 互动表现 |
|------|------------|---------|
| 7:00-9:00（早通勤） | 资讯、短干货 | 阅读量高，互动一般 |
| 12:00-14:00（午休） | 轻松向、故事 | 阅读量中，互动较好 |
| 18:00-20:00（晚通勤） | 教程、深度 | 阅读量中，互动好 |
| 20:00-22:00（睡前） | 情感、深度长文 | 阅读量高，互动最高 |
| 22:00-23:00（深夜） | 情感、反思 | 互动率高但受众窄 |

### 内容矩阵健康度

健康的内容矩阵应满足：
- **主题聚焦度** > 70%（核心主题占比）
- **原创比例** > 60%（转载过多削弱账号价值）
- **内容类型多样性**：至少覆盖 3 种以上内容类型
- **爆款占比**：10-20% 的文章贡献 60%+ 的阅读量属正常分布
