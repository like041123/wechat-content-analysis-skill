# 报告模板与输出规范

本文件提供微信公众号内容分析的各类输出模板，供生成报告时参照。根据分析场景选择对应模板。

## 模板一：单篇文章分析报告

适用于单篇文章的深度质量评估。

### 报告结构

```
# 文章分析报告

## 基本信息
| 项目 | 内容 |
|------|------|
| 标题 | {文章标题} |
| 公众号 | {账号名称} |
| 发布时间 | {YYYY-MM-DD HH:MM} |
| 字数 | {字数} |
| 预估阅读时长 | {分钟} |

## 执行摘要
{3-5 句核心结论，涵盖最大亮点与最大不足}

## 评分总览
| 维度 | 评分 | 等级 |
|------|------|------|
| 标题吸引力 | {X}/10 | {等级} |
| 结构逻辑 | {X}/10 | {等级} |
| 可读性 | {X}/10 | {等级} |
| 主题与情感 | {X}/10 | {等级} |
| 综合评分 | {X}/10 | {等级} |

## 详细分析

### 1. 标题分析
- **类型**：{标题类型}
- **评分**：{X}/10
- **亮点**：{具体亮点 + 原文引用}
- **不足**：{具体不足 + 原文引用}
- **优化建议**：{可执行建议}

### 2. 结构分析
- **开头策略**：{类型} —— {评价}
- **正文结构**：{评价}
- **结尾CTA**：{类型} —— {评价}
- **评分**：{X}/10
- **优化建议**：{可执行建议}

### 3. 可读性分析
- **平均句长**：{字数}
- **段落质量**：{评价}
- **图文配比**：{评价}
- **评分**：{X}/10

### 4. 主题与情感
- **核心主题**：{主题}
- **情感倾向**：{积极/中性/消极}
- **评分**：{X}/10

## 优化建议（按优先级排序）
1. **{高优先级建议}** —— {原因} —— {预期效果}
2. **{中优先级建议}** —— {原因} —— {预期效果}
3. **{低优先级建议}** —— {原因} —— {预期效果}
```

## 输出格式选择指南

| 场景 | 推荐格式 | 推荐工具 |
|------|---------|---------|
| 正式汇报/分享 | HTML 可视化报告 | `html-report` 技能 |
| 数据整理/后续处理 | Excel 表格 | `xlsx` 技能 |
| 快速结论/日常沟通 | Markdown 对话内输出 | 直接输出 |
| 策略汇报/培训 | HTML 演示文稿 | `html-deck` 技能 |
| 多维度数据可视化 | HTML 报告含图表 | `html-report` + ECharts |

## HTML 报告设计要求

当使用 `html-report` 生成可视化报告时，需满足：

1. **字体**：中文使用 `Noto Sans CJK SC` 或 `WenQuanYi Micro Hei`
2. **配色**：专业商务风，主色推荐深蓝/墨绿，辅以暖色高亮关键数据
3. **图表**：
   - 评分维度使用雷达图
   - 主题分布使用饼图/环形图
   - 时间趋势使用折线图
   - 多账号对比使用分组柱状图
4. **响应式**：支持桌面与移动端查看
5. **可读性**：表格交替行底色、关键数据加粗高亮

## 注意事项

- 报告中所有评分必须有证据支撑，不得仅给分不给理由
- 优化建议必须可执行，包含具体操作步骤
- 对比报告中避免主观贬低任一账号，保持客观中立
- 涉及数据的部分需标注数据来源与时间范围
- 报告末尾附"分析局限说明"，提示数据样本量与时效性限制
